# EMFF-IFDGCN-with-Layer-Adaptive-Pruning-for-Alzheimer-s-Disease-Classification

Alzheimer’s Disease (AD) is a neural disorder in the brain identified as dementia. This neurological disease affects the brain functions. The AD is diagnosed using the Magnetic Resonance Imaging and neuropsychological examinations in manual analysis. Deep Learning, and Artificial Intelligence introduced new techniques for automating medical image diagnoses. Eventhough, advancements in the medical field, still faces challenges like ignoring features in specific regions of interest, necessitates retraining each time when increasing image, and less suitable for hardware implementation. Hence, this study proposed an Enhanced Multi-scale Feature Fusion Network to fuse the multi-modal data for Integrated Fusion Dual Graph Convolution Network based AD classification with Layer-Adaptive Magnitude-based Pruning. A widely recognized resource for AD classification through MRI_ PET_data leveraging various performance metrics of the proposed approach achieving highest Accuracy of 96.67% outperforms existing approaches, demonstrating its potential as an advanced tool for accurate and interpretable AD diagnosis.

# Proposed Methodology

AD is a neural disease which affects about 50 million people worldwide. The disease causes an irreversible damage to the brain that affects cognition, memory, and other functions. The primary intention of this network is to fuse multi-modal data using an EMFFN and GCN-based AD classification with LAMP. Initially, the input MRI and the PET image data are obtained from the MRI_PET_data dataset. The acquired image samples are categorized into spectral and spatial features, and it is delivered to the fusion, in which the fusion of the multi-modal data is done by using the EMFFN. In the fusion network, it combines the preliminary decisions made in each branch and produces a final identification. Then the fused multi-modal data are delivered into a classification phase in which the various classes in the AD are categorized using the IFDGCN. The GCN layer defines the layers to be used in the network, and it generally involves an input layer, which is used for accepting the graph-based representations. The multiple convolution in the GCN is used for feature learning and the output layer at the end of the GCN is used for classification. In the classification phase the GCN has a lot of parameters and it was found as difficulty in the classification process so that a pruning approach is introduced which is used for making the GCN as a lightweight architecture. The LAMP score avoids heavy computation or hyperparameter tuning integrating the model-level 2 distortion brought by pruning. Figure 1 demonstrates the architectural diagram of the proposed model.
